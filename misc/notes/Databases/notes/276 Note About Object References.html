In the next lecture you'll be learning around Object References with MongoDB and Mongoose.

Some of the syntax has changed in the latest versions of MongoDBand Mongoose.

Instead of using foundUser.posts.push(post)you will use foundUser.posts.push(post._id)

These changes are for MongoDB 3.6.1 and Mongoose 5.0.0-rc2 (which you should be using).

You can check/update your version numbers by typing the following commands in the terminal:

mongo --versionand

npm uninstall mongoose
npm install mongoose@5.0.0-rc2

The first command will return your mongodb version and the second two will automatically update the mongoose version.

If you need to update your mongodb version to 3.6.1 then gohere and skip to theUpdatemongodb to version 3.6.1 step.
